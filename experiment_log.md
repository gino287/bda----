## v1.0　基礎版 Baseline

### 設定

| 項目 | 內容 |
|------|------|
| 文章篩選 | LIKE 關鍵字比對（公司名稱 + 股票代號） |
| 前處理 | N-gram（2~4 gram）+ 過濾純數字/標點片段 + DF >= 5 |
| 向量空間 | TF-IDF × Chi-square，TOP_K = 200 |
| 分類方式 | 單一分類器 |
| 回測窗口 | 30 天滾動 |

### 分類結果

| 分類器 | 準確率 |

| Naive Bayes | **57.3%** |
| Decision Tree | 53.8% |
| Random Forest | 51.6% |
| KNN (k=5) | 49.2% |

## 待測試方向

| 優先 | 方向 | 預期改善 |
|------|------|----------|
| 1 | TOP_K 調大（500 / 1000） | 特徵太少可能丟失資訊 |
| 2 | 換斷詞（monpa / ckip） | N-gram 語意不準 |
| 3 | 只用新聞來源（Yahoo） | 混合來源互相干擾 |
| 4 | 調整 N / SIGMA 參數 | 觀察不同預測窗口的影響 |
